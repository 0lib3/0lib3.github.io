Metaverse Token Economy Analysis

An end-to-end data analytics pipeline investigating whether metaverse crypto gaming tokens are viable for in-game economies. Starting from 347 raw CSVs, I built a full data warehouse, wrote analytical SQL, and published a business intelligence dashboard.

347 Tokens Analyzed
124K Rows Cleaned

Python PostgreSQL pandas CTEs Window Functions ETL Pipeline Metabase GitHub
